Instructions:
Hold space to see the clipboard and relevant progress in completing the game. (Let go and the
clipboard disappears.)
Left-click on an object to select it for npc motion and mesh combination. You can only select
objects that are highlighted as green with the pointer/ line renderer.
Right-click on an object to rename it. (can rename any object the pointer can land on)
Once done typing the new name, press Enter.
If you decide not to rename the object, press Escape.
Move around with w, a, s, and d (or the arrow keys).
